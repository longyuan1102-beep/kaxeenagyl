## 🎯 启动步骤（必读）

### 步骤1：启动后端

**新开一个 PowerShell 窗口**，运行：

```powershell
cd F:\自建程序源码\kaxeenagyl\server
npx nest start --watch
```

**等待看到**：
```
🚀 服务器运行在 http://localhost:3001
```

---

### 步骤2：启动前端

**再开一个 PowerShell 窗口**，在项目根目录运行：

```powershell
cd F:\自建程序源码\kaxeenagyl
npm run dev
```

**等待看到**：
```
Ready on http://localhost:3000
```

---

### 步骤3：访问系统

浏览器打开：**http://localhost:3000**
